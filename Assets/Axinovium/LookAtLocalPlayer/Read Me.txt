🇬🇧 Axinovium LookAtLocalPlayer – UdonSharp Script

Description
A lightweight UdonSharp script that makes any object automatically rotate to face the local player in VRChat.
Perfect for signs, characters, and interactive props.

How to Use

Add LookAtLocalPlayer.cs to any GameObject in your scene.

Adjust options in the Inspector:

Facing Axis – choose which side points toward the player.

Custom Rotation Offset – fine-tune the rotation angle.

Flatten Y / Smooth / Distance Limit – control movement and performance.

Press Play — the object will smoothly rotate toward the player.

Compatibility
✅ Works with VRChat SDK3 (Worlds) and UdonSharp.
✅ Optimized for both VR and Desktop modes.

🇯🇵 Axinovium LookAtLocalPlayer – UdonSharp スクリプト

説明
オブジェクトを自動的にローカルプレイヤーの方向へ回転させる軽量なUdonSharpスクリプトです。
看板、キャラクター、インタラクティブな小物などに最適です。

使い方

シーン内の任意のGameObjectに LookAtLocalPlayer.cs を追加します。

Inspectorで設定を調整します：

Facing Axis – プレイヤーに向けたい軸を選択

Custom Rotation Offset – 角度を微調整

Flatten Y / Smooth / Distance Limit – 動きやパフォーマンスを制御

再生ボタンを押すと、オブジェクトがプレイヤーの方を向くようになります。
